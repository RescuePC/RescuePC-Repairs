=============================================
   RescuePC IMAGES - README
=============================================

This directory should contain:
1. RescuePC_Logo.png (main logo)
2. RescuePC_Logo.ico (icon for Windows)
3. RescuePC_Banner.png (header image)
4. Tool icons (if needed)

Branding guidelines:
- Primary color: #ff9900 (orange)
- Background: #222222 (dark gray)
- Text: #FFFFFF (white)
- Button bg: #333333 (medium gray)
- Pro elements: #ff9900 (orange)

The logo and icons should be designed to be recognizable
even at small sizes and should work well on dark backgrounds.
